void FBO::ProcessDrawTex ( std::shared_ptr< Program > shader,
GLuint inTex )

Bind this FBO.

This assumes that shader is setup prior to have the "texBuf" uniform, where the passed texture data will be mapped to that uniform.

If inTex is 0, then we don't bind any texture. This is useful for writing the first time, without any texture data generated yet.
